Apa itu If?

If berasal dari bahasa Inggris yang berarti 'Jika'.
Dalam dunia programming, If adalah statement untuk mengatur alur jalannya suatu program, jika kondisi seperti ini maka lakukan ini, dan jika kondisi seperti itu maka lakukan hal tersebut.

Contoh penulisan sintaks-nya adalah seperti dibawah ini:

If (kondisi == true) {
lakukan perintah
} else {
lakukan perintah lain
}

Contoh kasus

1. Buatlah program yang mencetak "Angka genap" jika user memasukan angka genap, begitu pula sebaliknya.
2. Buatlah program syarat naik hysteria, contoh tinggi minimum & umur.

Keterangan

Output Program 1 & 2 bergantung pada inputan user.
Inputan yang berbeda akan menghasilkan hasil yang berbeda.

Contoh program If:

Program ganjil genap
Program naik Hysteria

Apa itu looping?

Looping berasal dari bahasa Inggris yang berarti 'Putaran'.
Dalam dunia pemrograman, looping sendiri adalah sebuah statement yang menjalankan sebuah perintah berulang kali selama kondisi yang ditentukan bernilai True atau sampai kondisi tertentu.

Contoh penulisan sintaks untuk looping jenis for loop:

for(int i = 0; i < kondisi; i++) {
perintah yang akan dijalankan berulang
}

Contoh kasus

1. Buatlah program faktorial berdasarkan input dari user!
2. Buatlah program yang mencetak segitiga siku-siku berdasarkan input dari user!

Keterangan

1. Program 1 menggunakan rumus faktorial yaitu n! = n x (n - 1) ... x 1 dalam looping
2. Program 2 merupakan setengah pyramid.

Apa itu Array?

Array adalah struktur data yang dapat menampung banyak kumpulan data.
Array juga menggunakan indeks untuk memudahkan akses terhadap data yang disimpan.
Indeks array dimulai dari angka 0.
Sedangkan looping array adalah statement atau proses mengiterasi sebuah array.

Cara deklarasi sebuah array:

tipeData namaArray [ukuranArray];
e.g. int kelas [5];

Contoh kasus

1. Buatlah program untuk menghitung berapa banyak angka ganjil dalam sebuah array!
2. Buatlah program array daftar belanja berdasarkan inputan user!

Keterangan

1. Program 1 menambahkan 1 nilai terhadap variable genap setiap bertemu angka genap dalam array.
2. Program 2 menambahkan isi dari berdasarkan inputan user.

Apa itu Procedure?

Procedure adalah sebutan untuk fungsi yang tidak mengembalikan nilai void.
Tujuan dari procedure adalah untuk menyederhanakan struktur kode / manajemen kode.
Biasanya procedure berisi rumus, hitungan, laporan dll.

Cara pembuatan procedure / fungsi:

static tipeData namaFungsi() {
isi fungsi
}

Contoh kasus

1. Buatlah program perkalian sederhana berdasarkan inputan user!
2. Buatlah program hitung luas lingkaran berdasarkan inputan user!

Keterangan

1. Program 1 mengkalikan dua angka berdasarkan inputan user
2. Program 2 menggunakan rumus luas lingkaran L = pi x r2
